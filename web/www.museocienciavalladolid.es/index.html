¿Qué son y por qué son importantes las conexiones seguras (HTTPS)?

  • 🔒 HTTPS permite a las webs cifrar todo el contenido mientras viaja de la web a tu dispositivo y vuelta, pero este debe estar bien implementado.
  • 🕵️ Una web con un HTTPS bien implementado impide que nadie pueda interceptar los datos que introduces en formularios, incluidas contraseñas, datos bancarios, personales o médicos.
  • 💉 También impide que actores malintencionados "inyecten" contenido malicioso o modifiquen la información de forma no detectable en una web.
  • ⚠️ Un sitio con una puntuación C o menor, implica que tiene una implementación deficiente y no es completamente seguro, facilitando que tu privacidad usándolo esté en riesgo.
